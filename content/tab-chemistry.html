Vanliga kemiska formler

Ainemäärä = \(\frac{\text{Massa}}{\text{Moolimassa}}\) Substansmängd = \(\frac{\text{Massa}}{\text{Molimassa}} \(n = \frac{m}{M}\)
Kaasun ainemäärä = \(\frac{\text{Tilavuus}}{\text{Moolitilavuus}}\) Gasens substansmängd = \(\frac{\text{Volym}}{\text{Molvolym}}\) \(n = \frac{V}{V_\mathrm{m}}\)
Ainemäärä = \(\frac{\text{Hiukkasmäärä}}{\text{Avogadron vakion}}\) Substansmängd = \(\frac{\text{Partikelantal}}{\text{Avogadros konstant}}\ \(n = \frac{N}{N_\mathrm{A}}\)
Konsentraatio = \(\frac{\text{Ainemäärä}}{\text{Tilavuus}}\) Konsentration = \(\frac{\text{Substansmängd}}{\text{Volym}}\) \(c = \frac{n}{V}\)
Tiheys = \(\frac{\text{Massa}}{\text{Tilavuus}}\) Densitet = \(\frac{\text{Massa}}{\text{Volym}}\) \(\rho = \frac{m}{V}\)
Kaasun tiheys = \(\frac{\text{Moolimassa}}{\text{Moolitilavuus}}\) Gasens densitet = \(\frac{\text{Molmassa}}{\text{Molvolym}}\) \(\rho = \frac{M}{V_\mathrm{m}}\) (NTP)

Sähkömäärä elektrolyysissä Elmängd i elektrolys \(Q = It = nzF\)
